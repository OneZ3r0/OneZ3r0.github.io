与反序列化的关系

Java反序列化时，会调用被反序列化的readObject方法，当readObject方法书写不当时就会引发漏洞，我们通常会通过反射调用去绕过对我们的waf。或者如果我们在进行反序列化过程中可控的话，能够加载我们使用反射调用的恶意命令语句的话，则能够成功进行命令执行。
